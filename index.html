Documentations

Base URL : http://localhost:8000

GET / Statut de l'API

Vérifie la disponibilité du serveur et retourne ses métadonnées (version, uptime, nombre de lettres).

GET /letters Liste des lettres supportées

Retourne toutes les lettres de l'alphabet dactylologique avec leurs métadonnées.
Paramètre optionnel : ?type=voyelle ou ?type=consonne

GET /letters/{letter} Détails d'une lettre

Retourne les informations détaillées pour une lettre donnée.

POST /predict Prédire une lettre depuis une image

Envoie une image encodée en base64. Retourne la lettre détectée, la confiance et le top-3.

Corps JSON : {"image": "<base64>", "format": "jpeg"}

GET /stats Statistiques de session

Retourne les statistiques d'utilisation depuis le démarrage du serveur.

DELETE /stats Réinitialiser les statistiques

Remet à zéro les compteurs de session côté serveur.
