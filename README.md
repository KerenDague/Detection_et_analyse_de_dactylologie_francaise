# Détection et analyse de Dactylologie Française

## 🎯 Objectif 

Ce projet vise à développer un modèle d'apprentissage automatique capable de reconnaître les lettres de la dactylologie françasie (considérée commel'alphabet en Langue des Signes Française (LSF)) à partir d'images et de vidéos, puis de les traduire en français écrit.

## 🗃️ Données

**Nous avons construit notre corpus aussi les doonées ne sont elles pas exposées sur ce dépôt**

Le corpus contient :
- Des vidéos originales 
- Des vidéos issues de corpus préexistants
- Des vidéos produites pas augmentation de données
